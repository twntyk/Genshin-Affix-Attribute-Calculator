Formula for Minor Affix Probability

The probability for a specific minor affix is calculated as:

P(A) = Weight(A) / ∑(Weight of Available Minor Affixes)

Example Calculation

For example, if a 5-Star Plume of Death already has ATK%, Energy Recharge%, and CRIT Rate% minor affixes when it reaches level 4, the probability of obtaining CRIT DMG% as a 4th minor affix is:

P(CRIT DMG) = Weight(CRIT DMG) / (Weight(HP) + Weight(DEF) + Weight(HP%) + Weight(DEF%) + Weight(EM) + Weight(CRIT DMG))
           = 3 / (6 + 6 + 4 + 4 + 4 + 3)
           ≈ 11.1%
